Les navigateurs web

Un peu d'histoire

Le premier navigateur date de 1990. En 1993, le premier navigateur stable est diffusé en masse. Internet Explorer apparaît en 1995. À partir des années 2000, Internet Explorer devient le leader du marché en grande partie car, il est installé par défaut, sur tous les micro-ordinateurs sous Windows.

À partir de 2005, Mozilla FireFox commence à détrôner Internet Explorer, puis, à partir de 2012, c’est au tour de Google Chrome de devenir le navigateur le plus utilisé dans le monde. Microsoft Edge remplace Internet Explorer en 2015 et depuis 2020 ils est construit sur la même base Chromium que Google Chrome.

Opera existe depuis 1994 et se maintient notamment grâce à Opera Mini, le navigateur mobile.

Les principaux navigateurs

Les navigateurs listés ci-dessous représentent les plus grandes parts de marché. Certains ne fonctionnent que sur desktop (Internet Explorer) tandis que d'autres fonctionnent aussi bien sur desktop que sur mobile (Google Chrome, Safari...).

  • Google Chrome
  • Microsoft Edge / Internet Explorer
  • Mozilla Firefox
  • Apple Safari
  • Opera
